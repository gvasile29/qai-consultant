# Octoverse 2024: AI Leads Python to Top Language as Global Developer Community Surges

## Executive Summary

GitHub's 2024 Octoverse report reveals three major trends: a surge in global generative AI activity, rapid growth in developers worldwide (especially in Africa, Latin America, and Asia), and Python's emergence as the most-used language on GitHub, overtaking JavaScript after a decade.

## Key Statistics

- **518 million** total projects on GitHub with 25% year-over-year growth
- **Nearly 1 billion** contributions to public and open source projects
- **5.6 billion** contributions to all projects on GitHub
- **137,000** public generative AI projects with 98% year-over-year growth
- **1+ million** maintainers, teachers, and students using GitHub Copilot for free
- **5.2 billion** contributions to all projects in 2024
- **108 million** new repositories created in 2024
- **39 million** secret leaks detected via secret scanning
- **10.54 billion** GitHub Actions minutes used (30% year-over-year increase)

## Global Developer Community Growth

Developers made more than 5.2 billion contributions to over 518 million projects in 2024. Greater growth is occurring outside the United States every year since 2013, with this trend accelerating.

## Programming Languages

Python overtook JavaScript as the most-used language on GitHub for the first time after JavaScript's 10-year reign.

**Key Context:**

## Three Key Takeaways

1. **Generative AI models are becoming core building blocks** in software development, powering coding tools, vulnerability fixes, and learning experiences. Developers benefit from platforms enabling easy AI model experimentation as building blocks.

2. **Global developer community is expanding rapidly** with increasingly diverse contributors. Accelerated growth particularly evident in Africa, Latin America, and Asia. AI tools may lower entry barriers for new developers across regions.

3. **The definition of "developer" is evolving** beyond traditional software engineers to include operations and IT developers, machine learning researchers, data scientists, students, and teachers. This expansion reflects activity across data science, cloud infrastructure, and scientific research domains.

## Methodology

Data drawn from anonymized user and product activity October 1, 2023 through September 30, 2024. Only public activity included. Metrics reported only for economies with 100+ unique developers performing relevant activity. Full methodology available at press@github.com.
